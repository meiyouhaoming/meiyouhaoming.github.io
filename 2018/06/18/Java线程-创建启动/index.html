Java线程-创建启动

发表于 2018-06-18 | 分类于 Java |

Java使用Thread类代表线程，所有的线程对象都必须是Thread类或其他子类的实例

常用方法

static Thread currentThread()    //返回当前正在执行的线程的引用
String getName()    
void join()    //等待这个线程死亡
void join(long mills)    //等待最多mills毫秒
void run()    //用来包含在线程中执行的代码
static void sleep(long mills)    //使当前执行的线程休眠指定的毫秒数
void start()    //执行线程

继承Thread类创建线程类

继承Thread类并重写run() 方法即可实现一个线程
本例共有3个线程,主线程和程序显示创建的2个子线程，
不能共享资源

实现Runnable接口创建线程类

第一种方法创建的Thread子类即可代表线程对象，第二种方法
创建的Runnable对象只能作为线程对象的target,多个线程可以
共享一个target,所以可以实现资源共享
