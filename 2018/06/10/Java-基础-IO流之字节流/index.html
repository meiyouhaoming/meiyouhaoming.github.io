Java 基础 : IO流之字节流

发表于 2018-06-10 | 分类于 Java |

字节流

按照流的流向，可以将流分为输入流和输出流。若按照流的操作数据类型来分，可分为字节流和字符流。字节流可用于读写字节，通常可操作图片视频等数据。字符流可以读写字符数据，方便对文本文件进行操作。

InputStream

InputStream 是所有输入流的抽象基类，本身不能创建实例来执行输入，必须使用其子类来创建对象。

  • Java 7改写了所有的IO资源类，他们自动实现了AutoCloseable接口，因此都可以通过自动关闭资源的try语句来关闭这些IO流
